# CompanyApp

A web application built with ASP.NET Core MVC for managing company data.

## Technologies used
- C# / ASP.NET Core MVC (.NET 8)
- Entity Framework Core with SQLite
- Bootstrap 5
- Visual Studio 2022

## Features
- CRUD operations for Departments and Employees
- Responsive UI with Bootstrap 5
- SQLite database with seed data

## How to run
1. Open `CompanyApp.sln` in Visual Studio 2022
2. Run `Update-Database` in Package Manager Console
3. Press F5 to start the application
